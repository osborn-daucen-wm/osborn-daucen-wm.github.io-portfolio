My Languages

HTML

HTML is the basic layout of a web page. It is essential to moving forward when working in both the front and back end. It was my first language to learn and allows me to create basic, static websites.

CSS

CSS is essentially what makes a website look good. It is the styles that goes hand-in hand with your HTML. CSS was the second lenguage I learned. Currently, I am trying to expand by working into SASS.

JavaScript

JavaScript, not to be confused with Java, is the intelligence that is built into a website. It allows for something a little more dynamic as well as interactive. The uses for JavaScript are literally endless.




SQL

SQL is efficient for working with databases. This is part of the server-side of programming that pays significantly more. I have personally used it within my work on PHP; creating relational SQL databases.

PHP

PHP is the first truly back-end language. It is interweavable with HTML which allows for a lot less of a learning curve and is a key part when learning the LAMP stack. (the "P" is for PHP)

NodeJS

NodeJS is the back-end component that is key to the MEAN stack. It uses the same basic JavaScript syntax to run back-end functionality to a website. Specifically, it often helps connect to a database like MongoDB.




Java

My first encounter with Java was at my trade-school, West-MEC. But, I began gaining a deeper understanding of the language and how it worked in my Freshman year of college at ASU. Since then, I have grown fond of the language and even managed to get a job grading coding assignments and tutoring students in the class for an Intro to Programming class, CSE 110.
